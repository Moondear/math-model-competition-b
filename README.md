## 🚀 快速开始

### 方法1: 一键启动所有系统 (推荐) ⭐⭐⭐
```powershell
.\朋友启动三个Web系统.ps1
```
或者直接双击运行 `朋友启动三个Web系统.bat`

### 方法2: 单独启动模块
```python
# 轻量级仪表盘
python dashboard_lite.py

# 带安全特性的仪表盘
python dashboard_safe.py

# 交互式演示系统
python interactive_showcase.py
```

### 方法3: 运行具体算法
```python

# 可视化与报告生成

```
