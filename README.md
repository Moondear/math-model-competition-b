## ✨ 核心特色

### 🔬 数学建模算法
- **抽样检验优化** - 3D成本表面分析 + 增强OC曲线
- **生产决策优化** - 16种策略智能分析 + 清晰柱状图
- **多工序网络优化** - 立体3D流程图 + 箭头指示 + 决策状态
- **鲁棒性分析** - 蒙特卡罗模拟 + 置信区间 + 风险评估

### 🚀 创新算法 (5项)
- **量子启发优化** - 模拟量子退火机制
- **联邦学习预测** - 隐私保护的次品率预测
- **区块链供应链** - 防篡改决策记录
- **分布式求解** - 支持千万级变量
- **实时决策引擎** - 5秒更新周期

### 🎨 商业级可视化
- **3D立体图表** - 真正的三维数据展示
- **专业配色方案** - 现代UI/UX设计标准 (#2ECC71, #E74C3C, #3498DB)
- **交互式Dashboard** - 实时参数调节，无文字重叠
- **响应式布局** - 自适应多种屏幕尺寸

### 🤖 AI答辩系统
- **智能问答知识库** - 70+专业问题覆盖，Word文档导出
- **实时评分系统** - 多维度评价分析，即时反馈
- **弱点热力图** - 个性化改进建议
- **压力训练模式** - 模拟真实答辩环境

## 🚀 快速开始

### 方法1: 一键启动 (推荐) ⭐
```powershell
.\启动Dashboard.ps1
```

### 方法2: 轻量版Dashboard (最稳定) ⭐⭐⭐
```bash
streamlit run dashboard_lite.py --server.port 8502
```
**访问**: [http://localhost:8502](http://localhost:8502)

### 方法3: 原版Dashboard (需OR-Tools)
```bash
streamlit run src/dashboard.py --server.port 8501
```

### 方法4: 完整程序 (命令行)
```bash
python src/main.py
```
